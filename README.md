# Drum Abuse Vol.5: MIDI Goes Consumer

Affordable MIDI rhythm boxes: TR-505, RX-21, Casio MT, DDD-5

Single-shot drum samples from **18 drum machines** released 1986, extracted from the [Synth Abuse Drum Machine Sample Collection](https://www.youtube.com/watch?v=Ay-U9eYKmGA).

- **232 MB** (2713 samples, `.wav`, loops excluded)
- 26 instrument categories: Agogo, Bell, Bongo, Cabasa, Clap, Clave, Conga, Cowbell, Crash, Cuica, Cymbal, FX, Gong, Guiro, Hi-Hat Closed, Hi-Hat Open, Kick, Perc, Ride, Rim, Shaker, Snare, Tambourine, Timbale, Tom, Whistle
- Source: [Drive download](https://drive.google.com/file/d/1eacxX1GtQWqJP3dmZYbYvEPuGtxMtLff/view)
- Thanks to https://www.youtube.com/@synthabuse

## Files in this repo

- `pack.json` — pack metadata
- `index.json` — array of machine summaries
- `instruments/<slug>.json` — flat list of samples per instrument category, across all machines
- `machines/<machine_id>.json` — per-machine detail (sample sets, instruments, sample lists)
- `samples/<machine_id>/...` — the `.wav` files and cover images
